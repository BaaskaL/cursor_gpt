# AGENTS.md

Энэ файл нь Cursor/AI агентууд энэ репод ажиллахдаа дагах **хамгийн бага, тодорхой дүрэм** болон **ажиллуулах командуудыг** агуулна.

## Товч танилцуулга

- **Repo төрөл**: жижиг Python төсөл (одоогоор `README.md`, `test.py` байна).
- **Зорилго**: өөрчлөлт хийхдээ аль болох жижиг, ойлгомжтой diff гаргах.

## Ажиллуулах / шалгах

- **Python ажиллуулах**:

```bash
python test.py
```

- **Энгийн синтакс шалгалт (зөвлөмж)**:

```bash
python -m compileall .
```

> Тайлбар: Одоогийн `test.py` нь `print(test)` гэж байгаа тул `test` тодорхойлогдоогүй үед алдаа гарч магадгүй. Агент өөрчлөлт хийхдээ энэ төрлийн алдааг арилгах эсвэл зорилгыг нь тодруулж, боломжит хамгийн жижиг засварыг хийнэ.

## Кодын хэв маяг (Python)

- **String нийлүүлэлт**: `+` ашиглан залгахын оронд **f-string** ашиглана.
  - Жишээ: `f"{name}={value}"`
- **Тайлбар/нэршил**: хувьсагч, функцийн нэрийг ойлгомжтой, боломжтой бол snake_case.
- **Алдааны мессеж**: хэрэглэгчид чиглэсэн тайлбар хэрэгтэй бол богино, тодорхой бичнэ.

## Өөрчлөлт хийх зарчим

- **Хамгийн жижиг өөрчлөлт**: шаардлагатай минимум мөрийг өөрчил.
- **Ойлгомжтой commit-diff**: холбоогүй refactor-ыг нэг дор бүү холь.
- **Баталгаажуулалт**: боломжтой бол өөрчлөлтийн дараа дээрх командуудаар ажиллаж/шалга.

## Git / автомат ажиллагаа

- Агент нь **өөрөө `git commit`/`git push` хийхгүй** (орчин автоматаар зохицуулах боломжтой).
- Хэрэв хэрэглэгч тусгайлан хүсвэл хийхээс өмнө ямар команд ажиллуулах, ямар үр дагавартайг товч тайлбарлана.

## Аюулгүй байдал

- Нууц мэдээлэл (token, key, password) **репод нэмэхгүй**.
- Гадны dependency нэмэх шаардлагатай бол шалтгааныг тайлбарлаж, **хамгийн бага** dependency-г сонгоно.
